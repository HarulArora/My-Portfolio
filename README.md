# Portfolio — React

A bold, dark-themed developer portfolio built with React.

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start dev server**
   ```bash
   npm start
   ```
   Opens at http://localhost:3000

3. **Build for production**
   ```bash
   npm run build
   ```

## Customisation

- **Your name** → Search & replace `[Your Name]` / `YOUR NAME HERE` in `src/App.jsx`
- **Resume** → Replace `public/your-resume.pdf` with your actual PDF
- **Links** → Update email, GitHub, LinkedIn in `src/App.jsx` (contact section)
- **Projects** → Edit the `projects` array in `src/App.jsx`
- **Skills** → Edit the `skillData` array in `src/App.jsx`
- **Experience** → Edit the timeline section in `src/App.jsx`
- **Tech stack tags** → Edit the `card-tags` section in the hero

## Stack
- React 18
- Plain CSS (no extra libraries)
- Google Fonts: Bebas Neue, DM Mono, Fraunces
